 sublime-text-config
============

My configuration of [Sublime Text](http://www.sublimetext.com/3)

* AutoFileName
* CSScomb
* dired
* DocBlockr
* Emmet
* File History
* GotoSelection
* HTML-CSS-JS Prettify
* Keymaps
* Local History
* MoveText
* Sass
* SideBarEnhancements
* SublimeLinter
* SublimeLinter-jshint
* SyncedSideBar

## Keyboard shortcuts
* Open dired in the current directory: CMD + ALT + D
* Prettify the current file: CMD + SHIFT + H
