## Keyboard shortcuts
* Multi cursor: CMD + D
* Open dired in the current directory: CMD + ALT + D
* Prettify the current file: CMD + SHIFT + H
* Add comma at end of line (run eol_comma macro): , -> n -> ,
* Add semicolon at end of line (run eol_semicolon macro): ; -> n -> ;
